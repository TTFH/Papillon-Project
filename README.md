### ACI stub server

The Ace Combat Infinity (ACI) servers were shut down on March 31, 2018. After 8 years the game is playable again thanks to community efforts.

## The Stop Killing Games Initiative

Publishers are permanently destroying the video games you've paid for. An increasing number of games are sold as goods, but designed to be completely unplayable for everyone as soon as support ends.

**Read more:** https://www.stopkillinggames.com/
